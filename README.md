Python Calculator 
======

Evaluates mathematical expressions as strings.

## Example:

```python
  t = evaluate('(4 + 4) * 344 + (((6 + 7) * 1333) + 2 + 100000) * (30 + 2)')
```

## Supported operations:

|Operatio|Symbol|
|:---:|:---:|
|Addition|+|
|Subtraction |-|
|Multiplication |*|
|Division|/|
|Exponentiation|^|
|Grouping (order of precedence) |( )|
